# Product Context: CalcuLabs

## Why CalcuLabs Exists

CalcuLabs é um portal brasileiro de calculadoras online gratuitas que ajuda brasileiros a tomar decisões em finanças, saúde, engenharia e ciência através de cálculos precisos e educativos.

## User Intent Model

Cada calculadora representa um nível diferente de intenção do usuário:

| Nível | Intenção | Exemplo |
|-------|----------|---------|
| Informacional | Usuário está aprendendo | Artigos do Knowledge Hub |
| Prática | Usuário quer calcular algo | Juros compostos, IMC |
| Comercial | Usuário está comparando | Financiamentos, investimentos |
| Transacional | Usuário está pronto para decidir | Simulação de aposentadoria |

## Page Value Hierarchy

### Alta Valor
- Calculadoras de financiamento
- Simuladores de investimento
- ROI / ROAS
- Simulações de aposentadoria

### Baixo Valor
- Matemática básica (MMC, MDC)
- Conversores simples

## Monetization Strategy

| Funil | Foco | Monetização |
|-------|------|-------------|
| Topo | Tráfego | AdSense leve |
| Meio | Engajamento | AdSense otimizado |
| Fundo | Conversão | Afiliados + Híbrido |

## Data Feedback Loop

```
Interação do usuário → Tracking (GTM + API) → Análise → Otimização do produto
```

## Strategic Principle

Cada página deve evoluir de uma calculadora simples para uma ferramenta de decisão.

O objetivo não é apenas uso, mas influência nas decisões do usuário.
