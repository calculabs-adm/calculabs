# Framework de Qualidade de Conteúdo — Google Standards

## Critérios de Qualidade do Google (2026)

### 1. E-E-A-T (Experience, Expertise, Authoritativeness, Trustworthiness)
- **Experience**: Conteúdo baseado em experiência prática
- **Expertise**: Demonstra conhecimento profundo do assunto
- **Authoritativeness**: Autoridade reconhecida no nicho
- **Trustworthiness**: Fontes confiáveis, dados verificáveis

### 2. Relevância e Utilidade
- Responde diretamente à intenção de busca do usuário
- Fornece valor prático e acionável
- Antecipa dúvidas e objeções do leitor
- Resolve problemas reais

### 3. Estrutura e Navegabilidade
- H1 único e descritivo
- H2/H3 hierárquicos e lógicos
- URLs amigáveis e descritivas
- Navegação interna consistente
- Links contextuais relevantes

### 4. Profundidade e Abrangência
- Tópico principal completamente explorado
- Subtópicos relevantes cobertos
- Múltiplas perspectivas apresentadas
- Exemplos práticos e aplicações reais
- Dados estatísticos e evidências

### 5. Originalidade e Criatividade
- Conteúdo único, não copiado
- Ângulos únicos e insights originais
- Linguagem natural e conversacional
- Elementos visuais originais

### 6. SEO Técnico
- Title tag otimizado (50-60 caracteres)
- Meta description atraente (150-160 caracteres)
- Schema markup estruturado
- Core Web Vitals otimizados
- Mobile-first design

### 7. Engajamento e Usabilidade
- Tempo de leitura estimado
- Elementos interativos (calculadoras)
- CTAs estratégicos
- Compartilhamento social facilitado
- Design responsivo

## Métricas de Qualidade por Tipo de Artigo

### Artigos Educacionais (Como os nossos)
| Métrica | Básico | Bom | Excelente |
|---------|--------|-----|-----------|
| Word Count | 800-1500 | 1500-2500 | 2500+ |
| H2/H3 Sections | 3-5 | 6-10 | 10+ |
| Internal Links | 2-3 | 4-6 | 7+ |
| External Links | 1-2 | 3-5 | 6+ |
| Images/Tables | 1-2 | 3-5 | 6+ |
| FAQ Items | 3-5 | 6-8 | 9+ |
| Social Proof | Nenhum | 1-2 | 3+ |

### Checklist de Qualidade Pré-Publicação

#### Conteúdo
- [ ] Tópico principal 100% coberto
- [ ] Pelo menos 3 exemplos práticos
- [ ] Dados estatísticos incluídos
- [ ] Fontes citadas e verificáveis
- [ ] Linguagem clara e acessível
- [ ] Sem erros gramaticais

#### Estrutura
- [ ] H1 único e descritivo
- [ ] Pelo menos 8 seções H2
- [ ] Hierarquia lógica H2→H3
- [ ] Introdução atraente (2-3 parágrafos)
- [ ] Conclusão que resume pontos-chave

#### SEO
- [ ] Title tag otimizado
- [ ] Meta description atraente
- [ ] URL amigável
- [ ] Palavras-chave naturalmente inseridas
- [ ] Alt text em todas as imagens

#### Engajamento
- [ ] FAQ abrangente (5+ questões)
- [ ] How-to guide prático
- [ ] Links para calculadoras relevantes
- [ ] CTAs estratégicos
- [ ] Elementos visuais (tabelas, listas)

#### Autoridade
- [ ] Dados de fontes confiáveis
- [ ] Citações de especialistas
- [ ] Comparações com concorrentes
- [ ] Evidências científicas quando aplicável

## Estratégias de Melhoria

### 1. Aumentar Profundidade
- Adicionar estudos de caso detalhados
- Incluir análises comparativas
- Explorar controvérsias e debates
- Adicionar contexto histórico avançado

### 2. Melhorar Estrutura
- Criar seções de "Mitos vs Fatos"
- Adicionar "Dicas Avançadas"
- Incluir "Casos Práticos"
- Criar guias passo-a-passo detalhados

### 3. Fortalecer Autoridade
- Citar fontes acadêmicas
- Incluir dados de pesquisa
- Referenciar especialistas
- Adicionar certificações ou credenciais

### 4. Otimizar para Usuário
- Adicionar navegação por âncoras
- Criar sumário interativo
- Incluir calculadoras inline
- Adicionar tooltips explicativos

### 5. Expandir Cobertura
- Abranger tópicos relacionados
- Criar séries de artigos conectados
- Adicionar conteúdo evergreen
- Explorar ângulos não cobertos

## Monitoramento e Métricas

### KPIs de Qualidade
- Tempo médio na página (>3 minutos)
- Taxa de rejeição (<40%)
- Posições no Google (TOP 10)
- CTR nos resultados de busca
- Backlinks de qualidade
- Social shares

### Ferramentas de Análise
- Google Search Console
- Google Analytics
- SEMrush/Ahrefs para SEO
- Hotjar para UX
- Screaming Frog para auditoria técnica
