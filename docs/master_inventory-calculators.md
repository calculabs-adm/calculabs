## Regras de Atualização

2. Toda remoção DEVE ser registrada (mover para seção "Removidas" com data)
3. Toda alteração de nome, slug ou tipo DEVE ser refletida na tabela correspondente
4. O campo ID corresponde ao `id` no `calculators.json`
5. O campo Slug corresponde ao `slug` no `calculators.json` e é usado na URL
